# 个人简历页面作业

## 作业要求：
1. **HTML 页面**：
   - 创建一个简洁的个人简历页面，包含以下部分：
     - 个人信息（姓名、职位、邮箱等）
     - 。。。（随意，如教育经历等；内容不是重点）
   
2. **CSS 样式**：
   - 使用 CSS 来美化页面，确保布局清晰，内容易于阅读。
   - 可以使用 `flexbox` 或 `grid` 布局。
   - 尽量保持设计简洁，避免过多花哨的效果。

## 提交要求：
- 在 `index.html` 文件中完成页面的结构和内容。
- 在 `style.css` 文件中完成页面的样式。

## 评分标准：
- 页面结构的清晰性和内容的完整性。
- 样式的简洁性和视觉效果。
- 代码的整洁和规范。
